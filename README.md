### Prerequisites

Libraries required:

* BOOST 1.63+ (libboost-all-dev)
* TBB 4.3+ (libtbb)
* OpenMP 3.x+
* GCC 7.1+
* Make 4.1+
* Doxygen 1.8.13+ (optional)

Installation in Debian based distros:

apt-get update && apt-get upgrade && apt-get install build-essential libboost-all-dev libtbb-dev
